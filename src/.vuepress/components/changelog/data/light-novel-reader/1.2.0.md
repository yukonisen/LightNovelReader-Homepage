# Changelog

## [1.2.0] - 2026-04-05

### 新增

- 创建插件系统：插件管理器、插件加载逻辑、APK 插件加载支持与 LNR API。
- 为 LNR API 开放更多接口，支持插件依赖注入、自定义插件主页内容与 Context 注入。
- 实现基础插件仓库功能与内容控件化系统。
- 添加数据源请求优先级与请求限制 API。
- 阅读器插图增加加载错误提示与重试按钮。
- 为 Android 9+ 设备支持 Edge to edge 效果。
- 设置页面新增开放源代码许可屏幕。
- 为 Android 9 及以下设备支持系统导航栏颜色适配。
- 重构阅读中页面「继续阅读」控件，支持显示最近三本书籍和快速章节选择。
- 新增数据源切换页面与数据源设置项按钮。
- 为扩展插件等页面更新英语和繁体中文本地化。
- 为 wenku8 数据源添加自动数据源调度器。
- 插件安装、更新、卸载流程改为状态机，插件安装对话框与 UI 状态解耦。
- 新增插件元信息缓存、错误插件自动禁用、本地插件应用列表。
- 支持从本地安装的应用扫描插件并加载。
- 支持禁用 API 不兼容的插件并显示提示。
- 新增 Designer 主题。
- 添加日活统计。
- 支持 LNR API 更新检查。
- 披露应用统计收集信息列表。
- 添加翻页模式点击中间展开菜单。
- 添加章节切换文本提示。
- 新增日期和时间格式设置。
- 新增深色模式独立阅读背景颜色。

### 变更

- 完善插件操作逻辑和 UI，并精简相关代码。
- 不再提供默认注解 API 版本值。
- 默认开启无限滚动。
- 更换网络请求库。
- 重构搜索 API、探索展开页面 API 与探索页面 API。
- 重构无痕滚动逻辑。
- 将阅读进度改为按章节存储。
- 升级 AGP 至 9.0。
- 重构数据源导出系统。
- 修改全局数据实现。
- 修改数据源 API。
- 重构插件加载系统。
- 重构应用图标与 SplashScreen。
- 重构更新对话框。
- 统计功能重构数据结构及页面。
- 迁移至 Coil 3。
- 重构插件详情页面。
- 设置页面重新排序。
- 优化书本详情页目录、UI 性能与 TopBar 效果。
- 微调软件整体设置项控件，优化主题设置页面显示效果。
- 优化章节进度保存逻辑与章节切换逻辑。
- 优化应用整体 Snackbar 效果。
- 优化阅读器沉浸模式、自动边距判断逻辑与阅读进度判断。
- 优化 AnimatedTextLine 显示效果。
- 统一应用书本 Skeleton 闪光效果，并增加平滑延迟进入动画以优化渲染性能。
- 更改阅读器背景图片显示模式为滚动模式专属，以防止视觉问题。
- 统一应用页面边距以适配各种设备导航栏。
- 调整阅读器阻止返回提示为 Snackbar 样式。
- 将书架页面 TabRow 调整为匹配内容宽度并居左显示。
- 优化应用文本和段落效果、章节选择底栏操作体验与整体字体版式。
- 优化图片查看器触发方式，移除缩放效果。
- 重构标记为已读对话框。
- 移除本地牛皮纸，使用在线背景资源。
- 启用优化资源缩减。
- 归零插件 API 版本号。
- 更改 Snapshot variant 的版本号格式为 a_b_c_SN。
- 更新相关域名。
- 为导入数据提供合并或覆盖选项。
- 优化在慢速网络情况下的书本导出稳定性。
- 优化阅读中页面可能频繁重组的问题。
