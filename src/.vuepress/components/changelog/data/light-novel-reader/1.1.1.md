# Changelog

## [1.1.1] - 2025-08-19

### 新增

- 重新支持隐藏阅读器电池指示器。
- 图片查看器支持保存和查看封面图片。
- 支持长按音量键翻页。
- 滑条支持长按输入自定义数值。
- 支持禁用手势返回。
- 完成文本格式化功能。
- 添加自动代理开启开关。
- 为文本格式化添加数据导出与导入支持。

### 变更

- 优化音量键翻页逻辑。
- 优化滑条设置项数值显示。
- 调整设置项显示。
- 优化 Wenku8 数据源请求标头。
- 重构文本后处理系统。
- 进行代码清理。

### 修复

- 修复图片加载指示器。
- 修正图片查看放大清晰度。
- 修复错误的重连机制。
- 修复导入大体量数据时请求频率过高的问题。
- 修复书本内容获取中错误的逻辑。
- 更新翻译与俄语本地化。
