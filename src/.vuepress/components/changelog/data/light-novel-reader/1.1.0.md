# Changelog

## [1.1.0] - 2025-07-13

### 新增

- 添加更多阅读器自定义项，包括字体颜色、字重、背景颜色和背景图片。
- 添加简繁转换功能。
- 添加已收藏书本图标提示。
- 添加书本详情界面中上次阅读书本标记。
- 添加缓存进度显示。
- 增加标签分类跳转能力。
- 阅读器支持手势放大图片。
- 完成分卷导出功能。
- 添加自动代理模块。

### 变更

- 重构更新检查。
- 更新截图与文档。
- 改进统计详情样式与活动部分展示。
- 改进 EPUB 导出时对失败的判定。
- 修正分卷导出的文件名与封面。
- 合并多语言本地化更新。

### 修复

- 修复阅读器图片解析问题。
- 修复错误主题。
- 更正书架导出行为。
- 修复自定义文本颜色相关问题。
- 修复初次阅读记录失效。
- 修复 WorkManager 在软件重启后自动恢复 work 导致的崩溃。
- 修复未支持字符的序列化错误。
- 修复翻页模式下进度保存失效。
- 修复主题适配问题。
