# Changelog

## [1.0.0] - 2025-02-07

### 新增

- 完成书架数据导出、全局数据导出与导入功能。
- 添加探索页下拉刷新功能。
- 添加多数据源切换功能，并缓存当前数据源数据。
- 添加批量添加书本至书架和书架分享功能。
- 书本操作加入触觉反馈，设置项新增图标支持。
- 重构探索主页、搜索结果和二级页面 UI。
- 阅读内容页设置栏新增分类设置项和实时预览。
- 更新渠道增加 GitHub 和加速地址支持。
- 完成 EPUB 导出功能。
- 完成“阅读中”页面和书本详情页新 UI。
- 新增应用关于提示、导出 EPUB 提示对话和通知。
- 增加 Android 15 支持。
- 页面动画支持。

### 变更

- 改进书架页、设置页、探索页、阅读内容页和详情页整体 UI。
- 压缩导出的数据文件。
- 调整点击翻页判定线。
- 优化选择对话框文本样式和多处 UI 细节。
- 重构导航系统，迁移全部 Dialog 实现至 Nav Dialog。
- 整体优化翻译。
- 调整阅读器默认字体和行距值。

### 修复

- 修复内容页错误下边距、滑动卡顿和章节名称过长显示问题。
- 修复错误上次阅读时间显示和阅读时间计数器。
- 修复自动更新检查中自动下载失败问题。
- 修复数据导出无法写入、导入失败提示和书架页面书本顺序颠倒问题。
- 修复其他界面切换到阅读中时会在空页和内容之间闪烁的问题。
- 修复快速滑动时对书本的误操作。
- 修复内容页滑动时卡顿、图片 URL 前存在空格时无法解析的问题。
- 修复 EPUB 导出无法触发的问题。
- 修复 Kotlin serialization proguard 规则、内容设置预览 Tab、对话框宽度、错误章节顺序等问题。
- 修复详情页返回阅读中页概率闪退、重复 popBackStack 跳转错误页面和阅读中页面载入问题。
