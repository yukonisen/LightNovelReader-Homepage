# Changelog

## [1.1.2] - 2025-08-27

### 新增

- 新增将现有全部章节标为已读功能。
- 为 Wenku8 数据源增加缓存机制。

### 变更

- 在导航前检查页面状态以提升 UI 稳定性。
- 优化详情页 UI。
- 优化统计书本信息获取逻辑。
- 移除“转至探索”按钮，改为提示。
- 优化返回至详情页的展示动画。
- 统一 DropDownMenu 字体字号。
- 更新俄语本地化字符串。

### 修复

- 修复自定义数值对话框概率无法获取值的问题。
- 修复搜索页加载提示显示。
- 修复格式化规则设置的一处错误。
- 修复阅读器翻页模式下点击翻页开关失效的问题。
- 修复标为已读后最近阅读章节为空的问题。
