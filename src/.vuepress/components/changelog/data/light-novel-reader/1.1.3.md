# Changelog

## [1.1.3] - 2025-09-06

### 新增

- 为书本卡片和详情页面添加加载动画。
- 为阅读中和书架页面增加懒加载。
- 新增 Snackbar 和部分提示信息。
- 阅读器支持过度滚动切换上下章。

### 变更

- 优化应用网络请求和性能，改善特定数据源访问稳定性。
- 进一步改善视觉效果。
- 改进书架页面稳定性。
- 优化书架书本信息加载逻辑和书架选择逻辑。
- 重构详情页的部分 UI。
- 更新 README 贡献说明。
