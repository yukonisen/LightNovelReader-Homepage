# Changelog

## [0.4.4] - 2024-09-17

### 新增

- 添加字数筛选器。
- 实现可滚动的添加至书架提示窗口。
- 添加点击翻页模式。
- 新增列表设置项已选项目文本动画。
- 新增书架分组折叠文本。
- 新增 Material You 动态配色。
- 添加快速切换章节功能。
- 增加翻页模式动画开关。
- 添加翻页模式图片加载支持。
- 添加自动屏幕边距检测和手动边距设置功能。
- 添加图片加载指示器、电量指示器、时间指示器、章节名称指示器和阅读进度指示器。
- 添加内容页文字选择支持。

### 变更

- 优化更新检查，改进更新逻辑并使用校验和检查文件。
- 将更新下载路径改到 cache/updates 目录。
- 优化开关设置项，新增禁用选项。
- 优化详情页、对话框和部分页面 UI。
- 优化应用主题应用逻辑。
- 重构 SplashScreen 图标，优化启动屏幕。
- 修改文字分页算法，自动去除每页上下空行并筛选空页。
- 调整探索展开页面缓存行为。

### 修复

- 修复搜索页使用手势返回后无限加载的问题。
- 修复检查更新渠道及日志。
- 修正对话框错误宽度与滑动指示器。
- 修复一些情况下更新时的闪退和异常。
- 修正书架页 Tab 宽度问题，并对 4 个以上 Tab 项使用滚动。
- 修复书架页 Tab 潜在闪退。
- 修复书架分组折叠图标方向错误。
- 修正 AppBar 折叠时点击编辑书架页面会显示错误 AppBar 高度的问题。
- 修复切换页面概率闪退和阅读中页面概率无法滚动问题。
- 修复无法正常恢复阅读进度问题。
